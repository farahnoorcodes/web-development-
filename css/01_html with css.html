three diff way to style

  1. inline css
  2. internal css
  3. external css

1. first one use style attributes like color, background-color, font in the tag you wanna style, this is called inline css.

2. Second method is to use style tag in the head tag of the html code , bye using this you don't have to add color and font for each tag or element .you can work in bulk and have large control on the website , or you can change things or style quickly. this is called internal css.

3. by using second method things get done but the size and length of the html file exceed . so we use an external css file to design the website , add attach it with a link of css file
