In HTML, you can add links, bold text, and italic text using specific tags.

to add a link just use the "a enclosed in angle brackets" tag with the href attribute.

href is the attribute that specifies the URL of the page the link goes to.

to bold text, use the "b enclosed in angle brackets< >" tag.

to italic text, use the "i enclosed in angle brackets< >" tag.

to underline text, use the "u enclosed in angle brackets< >" tag.
