form are used to make website interactive to the user.

User Interaction:

Tables and lists are for displaying data. Forms are the primary way to collect data from users (login, search, contact, registration).

Gateway to JavaScript:

Forms are the perfect bridge between HTML and your javascript folder. You can teach how to capture form data using JS.

Practicality:

It allows you to introduce new input types that are essential for modern web development.
