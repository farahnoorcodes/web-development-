you can add images to your webpage using the <img> tag.

you can add image to your web page from your local computer or from the internet.

to add image from your local computer, you can use the following code:

<img src="image.jpg" alt="description of image">
src is the path to the image file on your local computer. alt is used to provide a description of the image for accessibility purposes.

this is how i add an image from my local computer:
<img src="farah.jpg" alt="farah">

you can ajust the size of the image using the width and height attributes. for example:
<img src="farah.jpg" alt="farah Noor" width="500" height="300">
but for better practics, you should adjust only height or width at a time, and let the browser adjust the other dimension automatically to maintain the aspect ratio of the image.

for images from the internet, you can use the following code:
<img src="https://weresmartworld.com/sites/default/files/styles/full_screen/public/2021-05/aardbeien-3.jpg?itok=VKyyMjjg" alt="strawberries">
