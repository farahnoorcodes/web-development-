Id and Class

id:

  • Unique — only one element per page should have a given id
  • Selected in CSS with "#"
  • use for : Unique sections (nav, footer)
This is the header with id=header.

class:

  • Reusable — many elements can share the same class
  • An element can have multiple classes (space-separated)
  • Selected in CSS with "."
  • use for : Repeated styling patterns
